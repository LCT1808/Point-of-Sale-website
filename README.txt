MULTI-CATEGORY RETAIL STORE MANAGEMENT INFORMATION SYSTEM

Support tools required to run source code:
+ Microsoft visual studio
+ SQL server
+ Microsoft SQL Server Management Studio

Please follow the steps to run the source code:
- Open source code and build solution in Microsoft visual studio.
- Create a database in MSSMS, change connection string at "appsettings.json" file.
- Open Package Manager Console and "Update-Database".
- Open and run "initialization.sql" file to create admin account to access the system.
- Run the solution.
